# Tu Mejor Lista de Compra

## Descripción
"Tu Mejor Lista de Compra" es una aplicación diseñada para ayudar a los consumidores de la República Dominicana a ahorrar dinero en sus compras de supermercado. Al comparar precios de una amplia gama de productos en diferentes supermercados, nuestra aplicación te permite descubrir dónde puedes comprar tus artículos deseados al mejor precio.

## Características
- **Comparación de Precios:** Compara precios de productos en distintos supermercados de la República Dominicana.
- **Lista de Compras:** Crea y administra tus listas de compras para mantener un seguimiento de los productos que necesitas comprar.
- **Alertas de Ahorro:** Recibe notificaciones cuando tus productos favoritos bajen de precio en tus supermercados preferidos.
- **Fácil de Usar:** Interfaz amigable y navegación intuitiva para una experiencia de usuario óptima.

## Cómo Empezar
Para empezar a usar "Tu Mejor Lista de Compra", sigue estos pasos:

1. Clona este repositorio en tu máquina local usando `git clone URL_DEL_REPOSITORIO`.
2. Instala todas las dependencias necesarias ejecutando `npm install` en la raíz del proyecto (asegúrate de tener Node.js instalado previamente).
3. Una vez instaladas las dependencias, puedes iniciar la aplicación con `npm start`.
4. Abre tu navegador y visita `http://localhost:3000` para empezar a ahorrar en tus compras de supermercado.
